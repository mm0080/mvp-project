# WORK RULES (for AI agent)

## Goal
- Build an MVP in small, testable increments.
- Always keep the app runnable with `npm run dev`.

## Output rules
- Before editing: show a brief plan (files to change).
- Then apply changes.
- After changes: tell me exactly what to click/verify in the browser.

## Tech constraints
- Next.js App Router, TypeScript, Tailwind.
- UI text must be Korean.
- Keep components simple, avoid over-engineering.

## Scope discipline
- Implement only what is requested.
- If a missing decision is blocking, propose 2 options and pick the simplest by default.
